# Grading Rubric (interview-tough)

Grade every answer 0–10 across these, then give a single weighted score and a verdict.

| Dimension | What earns points | What loses points |
|---|---|---|
| **Correctness** | Factually right; no hand-waving | Wrong claims, vague gestures |
| **Depth (the *why/how*)** | Explains mechanism, derives, knows tradeoffs | Names the concept but can't go a level deeper |
| **Structure** | Clear, leads the answer, signposts | Rambling, no framework |
| **Tradeoffs** | Weighs alternatives, states assumptions | One-sided, no nuance |
| **Communication** | Crisp, uses correct terms, would convince an interviewer | Imprecise terminology |

## Verdict scale (be honest — this is the point)
- **9–10 ⭐** Interview-ready. An interviewer would be impressed.
- **7–8 🟢** Solid pass, minor gaps.
- **5–6 🟡** *Knows the what, not the why* — the trap. Mark for deep-dive, not a re-quiz.
- **3–4 🔴** Significant gaps. Re-teach before re-drilling.
- **0–2 🔴** Did not know it.

## Required feedback format after each answer
```
SCORE: X/10  [verdict emoji]
✅ What was strong: ...
❌ What an interviewer would push on: ...
🎯 The depth you're missing: ... (the specific deeper level)
💡 Model answer: ... (concise, the answer a strong Senior MLE gives)
📌 Remember this: <one line for revision>
```
Then: log it, update the profile, schedule review if score ≤ 6.

> Never inflate. A 6 is not a 7. The "I lack depth" problem is only fixed by honest 5–6 scores that trigger deep-dives.
