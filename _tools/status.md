# Tool: `status`

Show the skill profile + trend. Motivation + honest mirror.

## Behavior
1. Read `skill_profile.md` and `progress_log.md`.
2. Show:
   - **Pillar scorecard** — current level per pillar (count of ⭐/🟢/🟡/🔴).
   - **Trend** — last ~7 sessions: improving / flat / declining per topic touched.
   - **Top 3 strengths** and **Top 3 gaps** (lowest scores / most misses).
   - **Streak / volume** — sessions done, problems solved, mocks taken.
   - **Readiness estimate** — rough % to "Senior MLE interview-ready," with the 1–2 things moving it most.
   - **Due now** — items from the review queue.
3. End with the single highest-leverage next action.
4. Be honest about the readiness number — it should feel earned, not flattering.
